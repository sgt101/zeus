Ontology Options

  • Clicking the button will clear the current ontology from memory. You will be given an opportunity to save it before it is cleared.
  • Clicking the button will open a file requester enabling you to select which previously saved ontology to load into memory. If there is an existing ontology in memory you will be asked to confirm its deletion.
  • Clicking the button will save the current ontology to disk. If it is being saved for the first time a file requester will open enabling you to select its physical location.
  • Clicking the button will launch the Ontology Editor and create a new (empty) ontology if none is resident in memory.
