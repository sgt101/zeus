AgentViewer On-Line Help

Beside this window is a vertical toolbar consisting of several icons - representing the conduits, processors and databases of this agent. The conduits (inbox, outbox and message handler) facilitate the flow of information without altering them. The processors are the co-ordination engine and the planner & scheduler. The databases act as a repository of information for tasks, resources, ontologies, protocols and acquaintances. Selecting one of the icons will cause the associated window to open beside it, the options are:


Incoming mails .
This viewer displays in real-time this's agent's incoming mails.
Follow this link for more information.


Outgoing mails.
This viewer displays in real-time this's agent's outgoing mails.
Follow this link for more information.


Message handler.
This viewer displays the agent's how the agent responds to incoming mails.
Follow this link for more information.


Coordination engine.
This viewer displays goals being pursued by this agent and their composition.
Follow this link for more information.


Planner & Scheduler
This viewer displays the agent's diary of scheduled goals.
Follow this link for more information.


Acquaintance database.
This viewer displays an agent's relationships with other agents in the society and its belief about the capabilities of those agents.
Follow this link for more information.


Ontology database.
This viewer displays the facts known to this agents in terms of its legal attributes, the range of legal values for each attribute and any constraints between attribute values.
Follow this link for more information.


Protocols & Strategies.
This viewer displays the protocols and strategies used by this agent when communicating with other agents.
Follow this link for more information.


Resource database.
This viewer displays the resources that are owned by and available to this agent.
Follow this link for more information.


Task database.
This viewer displays tasks and their composition in terms of preconditions and effects.
Follow this link for more information.
