Replaying Saved Sessions

  • To replay a saved sessions the Visualiser must contact a DbProxy agent that has access to the saved file. See here for instructions on how to start a DbProxy agent.
  • Once the DbProxy is running a saved session can be loaded by selecting the Load Session option from the Playback menu.
  • A dialog will now appear asking for the identifying name of the Proxy agent that will provide the session; enter its name in the SelectDBProxy field.
  • The right-hand table will now list the saved sessions to which that Proxy has access. Select the session you want to restore and then click on the OK button.
  • Once the session has been restored the Visualiser will cease displaying live events and operate in an 'off-line' mode. The loaded session can then be replayed using the video control buttons or the options of the Playback menu.
  • The playback speed (the pause between screen updates during replay) can be changed through the Player Speed... option of the Options menu.
  • To revert to live visualisation, close the session using the Close Session option of the Playback menu.

The Video Control Buttons

Beginning - moves to the first frame of the session
Fast Rewind - steps backwards through the frames of the session
Rewind - slowly steps backwards
Prior Frame - moves back to the previous frame
Stop - halts any rewind or forward operation
Next Frame - moves forward to the next frame
Forward - slowly steps forwards
Fast Forward - steps forwards through the frames of the session
End - moves to the last frame of the session

If these buttons are 'greyed out' and unselectable this is because the Visualiser is still in the live playback mode. Once a saved session has been successfully restored these buttons will become selectable, and the record button will be greyed out.

Related Options

Instructions for saving a live session are listed here.

Other options from the Playback menu enable previously saved sessions to be deleted through the Delete Session option. Alternatively, all the saved sessions stored by a particular agent can be cleared by selecting the Purge Database option.
