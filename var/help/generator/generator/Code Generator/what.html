Code Generation

Note: As the information used to generate the agents is taken from the Agent Generator's internal model, you should ensure that changes to your project have been saved before attempting this stage.

The Code Generation window allows the developer to configure the utility agents, task agents through the following activities:

  • UTIL-1 : Configuring the Name Servers
  • UTIL-2 : Configuring the Facilitators
  • UTIL-3 : Configuring the Visualisers
  • UTIL-4 : Configuring the Database Proxies
  • TACF-1 : Configuring the Task Agents
These are described in more detail in Sections 5 and 6 of the ZEUS Realisation Guide.

Once configured, the scripts, source code and rules that implement the agent application can be generated. First choose where the source code will be written, and the operating system that will run the agents, then generate the application source code by clicking on the Generate button. Messages reporting on the code generation process will then appear in the "Generation Messages" panel at the bottom of the screen.
