Task Options

  • Clicking the button will create an instance of a new primitive task, summary task, rulebase or plan script.
  • Clicking the button enables you to edit the name of the currently selected task table entry.
  • Clicking the button will delete the currently selected entry from the project. You will be prompted to confirm deletion before it occurs.
  • Clicking the button will create a duplicate copy of the currently selected task table entry.
  • Clicking the button will launch the appropriate Task or Rule Editor window.
