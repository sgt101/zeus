Project Options

  • Clicking the button will clear the current project (including ontology) from memory. You will be given an opportunity to save your project before it is cleared.
  • Clicking the button will open a file requester enabling you to select a previously saved project definition file to load into memory. If there is an existing project in memory you will be asked to confirm its deletion.
  • Clicking the button will save the current project and ontology to disk. If it is being saved for the first time a file requester will open enabling you to select its physical location.
  • Clicking the button will launch the Society Viewer tool.
  • Clicking the button will launch the Code Generator, from where the agent source code can be generated.
