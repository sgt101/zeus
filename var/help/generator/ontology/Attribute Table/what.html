What is the Attribute Table?

The Attribute Table lists the attributes of the fact selected from the Fact Hierarchy panel above. There are two viewing modes:

Show Specific Attributes. This is the default option. When this is selected the table will show attributes which are specific to the currently selected fact alone, i.e. attributes inherited from the parent will not be shown.

Show All Attributes. When this option is selected the table will show every attribute possessed by the currently selected fact. Attributes shown in black are specific the currently selected fact; whilst attributes shown in red will have been inherited. Note that inherited attributes can not be edited.
