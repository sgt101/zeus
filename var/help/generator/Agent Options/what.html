Agent Options

  • Clicking the button will create an instance of a new agent.
  • Clicking the button enables you to edit the name of the currently selected agent, or the names of its tasks (if any).
  • Clicking the button will delete the currently selected agent from the project. You will be prompted to confirm deletion before it occurs.
  • Clicking the button will create a duplicate copy of the currently selected agent.
  • Clicking the button will launch the Agent Defintion window from where the agent's parameters can be configured.
