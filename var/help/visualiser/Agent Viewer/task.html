Task/Plan Database Viewer On-Line Help

What is it?
The Task/Plan Database Viewer is a component that displays the tasks and their composition in terms of preconditions and effects. The preconditions list the resources needed for execution of the task, while the effects list the resources that will be produced upon execution of the task.

How do I use it?

  • Select a task to view its preconditions and effects.
  • Select a precondition to view its attribute-value pairs.
  • Select an effect to view its attribute-value pairs.
