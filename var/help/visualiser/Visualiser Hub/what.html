Visualiser On-Line Help

Beside this window is a vertical toolbar consisting of several icons - this is the Visualiser Hub. Selecting one of the Hub's icons will cause the associated visualisation window to open beside it, the options are:


Society Viewer.
This viewer displays in real-time the known agents and their organisational relationships. It also shows the messages exchanged between the agents during problem solving.
Follow this link for more information.


Report Tool.
This tool shows the society-wide decomposition and distribution of active tasks, as well as the execution states of the various sub-tasks in real-time.
Follow this link for more information.


Agent Viewer.
This option launches a client-server version of the agent viewing utility. This will display the internal state of a chosen agent, regardless of whether the agent exists locally or remotely.
Follow this link for more information.


Control Tool.
This tool can be used to remotely review and/or modify the internal states of individual agents. It is effectively an online version of the Agent Generator enabling agent behaviour to be redefined at runtime by modifying the its task, resource, or organisational databases.
The control tool can also be used for administrative management, e.g. agents can be killed or suspended, and goals can be added or modified.
Follow this link for more information.


Statistics Tool.
This tool computes and displays individual agent and society-wide statistics, which can be presented in a variety of formats.
Follow this link for more information.
